Path Method Purpose Mongoose Method
Index /items GET List all items Item.find()
New /items/new GET Show new item form
Create /items POST Create a new item, then redirect Item.create()
Show /items/:id GET Show info about one item Item.findById()
Edit /items/:id/edit GET Show edit form for one item Item.findById()
Update /items/:id PUT Update one item, then redirect Item.findByIdAndUpdate()
Destroy /items/:id DELETE Delete one item, then redirect Item.findByIdAndRemove()
